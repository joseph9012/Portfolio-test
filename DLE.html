DLE

Project Overview

A personal project undertaken to improve the user experience of the Defence Learning Environment (DLE) platform, which is used by the British Army and me personaly. The aim was to create a more intuitive and user-friendly interface to access training materials and courses.

Timeframe

4 weeks

My Role

UX/UI Design, Visual design,
User flow, Prototyping, Research, Testing.

Tools

Figma, Zoom, Fig Jam.
The Problem
The website is very difficult to navigate and leads to unnecessary time loss.
The Aim
Create a website that allows a user to easily complete relevant course material.
As a personal user of this website I have heavily struggled to find courses that I must complete. As a reservist you must complete certain courses once a year, being part of the reserves you always see the group chat explode with confusion and issues relating to using the DLE website at the end of each year. Seeing how many people face problems with this website I decided to undertake this project.
The aim of this project is to improve the user experience of the website. Find the pain points and areas most in need of redesign via user research and feedback from current soldiers in the British army. I plan to analyse problems serving soldiers are currently facing as they try to complete courses.
I started with a user flow diagram which although looks very basic and easy to use I heavily struggled with the whole process. Being in the army I had seen multiple people stuggling with this process including me and was the basis for this project.
I had a look at issues I faced and peers said they faced. A big issue commonly found was how clustered the interface was and how difficult it was to navigate. I found lots of unnecessary courses on pages and the search function.
